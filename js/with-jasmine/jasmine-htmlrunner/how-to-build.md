# Jasmine & HTML Runner


## HTML Runner

- [jasmine-standalone-x.x.x.zip](https://github.com/jasmine/jasmine/releases)
- Unzip `jasmine-standalone-x.x.x.zip`


## Run tests

Open `SpecRunner.html` file in your browser.
